# 创新自检清单

写作前过一遍，确保至少一个维度跳出俗套。

---

## 一、烂大街论点黑名单（按主题分类）

遇以下主题时，避免直接套用括号内的常规角度，参考第三节找替代切入。

| 主题 | 烂大街论点（避免） |
|------|---------------------|
| 奋斗 | 奋斗是成功的唯一途径 / 奋斗的青春最美丽 |
| 传统文化 | 取其精华去其糟粕 / 传统文化是民族的根 |
| 科技/AI | 科技是一把双刃剑 / AI 既有机遇也有挑战 |
| 教育公平 | 教育公平是社会公平的基石 / 教育应该因材施教 |
| 环保 | 保护环境人人有责 / 绿水青山就是金山银山（非否定此论述，而是避免仅做注释式展开） |
| 诚信 | 诚信是做人之本 / 人无信不立 |
| 创新 | 创新是发展的第一动力 / 创新很重要 |
| 城乡差距 | 缩小城乡差距是当务之急 / 城乡应协调发展 |
| 互联网 | 互联网改变了人们的生活方式 / 网络是把双刃剑 |
| 读书 | 书中自有黄金屋 / 读书改变命运 |
| 坚持 | 坚持就是胜利 / 贵在坚持 |

**要点**：不是这些角度"错"，而是它们被写过太多次——阅卷人/读者看完三句就能猜完全文。除非你能给出全新的论据或推理路径，否则不要从这些角度切入。

## 二、烂大街论据黑名单

以下人物/案例在议论文中过度使用，除非找到新角度，否则避免作为主要论据：

| 类别 | 过度使用的人物/案例 |
|------|---------------------|
| 科学界 | 爱迪生（灯泡）、牛顿（苹果）、爱因斯坦（小板凳/相对论）、居里夫人（镭） |
| 历史界 | 司马迁（忍辱著史记）、屈原（投江）、岳飞（精忠报国）、勾践（卧薪尝胆） |
| 文学界 | 海伦·凯勒（假如给我三天光明）、保尔·柯察金、贝多芬（聋了作曲） |
| 商界 | 马云（创业）、乔布斯（辍学/斯坦福演讲）、任正非（华为）、马斯克 |
| 体育界 | 科比（凌晨四点的洛杉矶）、刘翔（退赛再起） |

**替代方向**：近三年新闻人物、学术研究中的具体发现、冷门历史事件、跨学科素材（认知科学、行为经济学、城市研究）。

**例外**：如果给出该人物的一个**全新切入角度**（如不是谈爱迪生"失败了一千次"，而是谈他的专利策略对创新的压制），可以用，但要注明新在何处。

## 三、创新角度示例库

每个常见主题给出 2–3 个非主流但站得住脚的切入角度。

### 奋斗
- **"奋斗的边际效用递减"**：不是反对奋斗，而是指出在特定条件下（行业天花板、制度瓶颈），追加奋斗的收益急剧下降，此时"换轨"比"加码"更理性
- **"被征用的奋斗叙事"**：当"奋斗"被某些力量用作剥削合法化的工具（996 是福报），需要区分真正的自我驱动与被迫超支
- **"不奋斗的权利"**：在一个把"闲"污名化的社会里，选择慢下来需要勇气——不是懒，是重新定义好生活

### 传统文化
- **"传统不是遗产，是工具箱"**：不是敬畏地保存，而是从传统中提取可用于当代问题的思维工具（如差序格局解释社交网络中的信任机制）
- **"被发明的传统"**（引用霍布斯鲍姆）：很多"传统"其实是近现代才建构出来的，真正的问题不是"要不要传承"而是"谁、为了什么、发明了哪些传统"
- **"低语境传统 vs 高语境传统"**：某些传统不需要解释就能跨文化传播（如中医针灸），另一些离了语境就失效（如特定仪式）——分类讨论而非笼统"传承"

### 科技 / AI
- **"AI 不是替代人，是重新定义人的比较优势"**：每次技术革命淘汰的不是人，是某种技能的市场价值——真正的问题是教育体系能否帮人持续迁移技能
- **"算法的合法性危机"**：当算法决定贷款额度、量刑建议、录取名单，问题不是"算法准不准"，而是"谁有权把规则写成代码"
- **"科技退步的正当性"**：在某些领域（如儿童产品、养老照护），"不够先进"反而是美德——不是所有场景都需要最高效的技术方案

### 教育公平
- **"公平 ≠ 同样对待"**：真正的公平可能要求对不同群体做不对等投入（给偏远地区卫星联网、给听障学生配助教）——"倾斜"不是不公平，是矫正存量不平等
- **"从机会公平到过程公平"**：底层家庭孩子进了好大学但文化资本不足（不知如何选课、利用导师时间），真正的公平要延伸到入学之后
- **"教育军备竞赛的双输结构"**：每个家庭单独加码是理性的，但所有人加码的结果是录取线抬高、童年消失、无人受益——需要从博弈论视角而非道德呼吁来理解

### 环保
- **"环保的阶级盲区"**：当环保政策抬高能源价格，买单的是低收入家庭——好的环保政策必须在设计阶段就纳入再分配机制
- **"自然的价值能否被定价"**：碳交易把排放权变成商品，但这套逻辑推到极致会要求给一棵树、一条河定价——有东西是"不应标价"的
- **"作为延迟惩罚的环保"**：不是"保护自然"（居高临下的主语），而是"把透支的账单还给下一代"（代际公平）

### 城乡差距
- **"城不是乡的终点"**：现代化 ≠ 城市化，乡村振兴不是把农村变成城市，而是在保留差异的前提下提高生活质量
- **"被忽视的城→乡流动"**：不只是农民工进城，也有城市知识青年下乡创业——双向流动中的新可能
- **"城乡之间的数字鸿沟是被低估的加速度"**：当城市在用 AI 辅导孩子做作业时，农村可能还在为网络信号发愁——差距在加速拉大而非缩小

### 诚信
- **"诚信不是道德自律，是重复博弈的理性策略"**：在一次性交易中失信有利，在重复博弈中诚信是占优策略——制度设计的任务是让交易变成重复博弈
- **"过度的诚信要求反而催生不诚信"**：当社会要求人在所有场合都绝对诚实（如要求绝症患者对家人"坦诚"），压力反而催生逃避——诚信需要允许合理的隐私和沉默

### 互联网
- **"互联网没有消灭中心，而是换了中心"**：去中心化是幻觉——流量仍然集中在少数节点，只是这些节点从报社变成了算法平台
- **"被互联网取消的'中场休息'"**：前互联网时代，信息传播有自然间隔（报纸一天一期），现在消息秒级推送——人在信息流中失去了暂停和反思的间隔

## 四、创新表达替换表

| 陈词滥调（砍） | 替代方向 |
|----------------|----------|
| 自古以来 | 删掉，直接说事 |
| 众所周知 | 删掉——如果真的众所周知，不需要说；如果不是，说了也没用 |
| 综上所述 | 换成具体的总结句（"从 X 到 Y，本文试图论证……"） |
| 随着……的发展 | 删掉，换一个具体时间或事件切入 |
| 在当今社会 | 删掉，直接用现象开篇 |
| 不可否认 | 换成具体的让步句（"X 确实……，但……"） |
| 在一定程度上 | 换成具体的程度描述（"在 X 条件下""当 Y 不存在时"） |
| 引起了广泛关注 | 换成具体数据或事件（"该话题在微博获得 X 亿阅读"） |
| 值得我们深思 | 删掉——全文本身就是深思，不需要宣告 |
| 路漫漫其修远兮 | 删掉（高考作文万能结尾重灾区），换自己的话收束 |
