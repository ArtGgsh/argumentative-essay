# 英文议论文结构模板

## 一、五段式（Five-Paragraph Essay）

适用：托福、雅思大作文、GRE Issue、一般学术英语写作。

### 结构骨架

| Paragraph | Name | Function |
|-----------|------|----------|
| 1 | Introduction | Hook + background + **thesis statement** (last sentence) |
| 2 | Body 1 | Topic sentence → evidence → analysis → transition |
| 3 | Body 2 | Topic sentence → evidence → analysis → transition |
| 4 | Body 3 | Topic sentence → evidence → analysis → transition |
| 5 | Conclusion | Restate thesis (in new words) + summary of points + final thought |

### 操作要点

- **Thesis statement** 是全文引擎：一句可被 challenge 的声明，不是事实陈述也不是预告。"This essay will discuss X" → 改为 "X should be reconsidered because A, B, and C."
- **Topic sentence** 是每段的论点，不是主题标签。不是 "First, let's look at economic factors" 而是 "Economic factors alone cannot explain the decline."
- **Transition** 要体现段落间逻辑关系（因果、递进、转折），不是机械的 "Secondly" / "Thirdly"。
- 结尾不要引入全新论点。Final thought 可以是一个 implication 或一个反问。

## 二、Toulmin 模型

适用：GRE Argument、高阶学术写作、需要严密逻辑的场景。

### 六要素

| Element | Chinese | Function |
|---------|---------|----------|
| Claim | 主张 | 你想让读者接受的核心观点 |
| Data / Grounds | 数据/依据 | 支持 claim 的事实或证据 |
| Warrant | 理据 | 连接 data 和 claim 的推理桥梁（往往隐含） |
| Backing | 支撑 | 当 warrant 本身需要论证时，提供额外支持 |
| Rebuttal | 反驳 | 承认并回应反方观点 |
| Qualifier | 限定 | 对 claim 的强度做限定（probably, in most cases） |

### 操作示例

> **Claim**: Stricter gun control reduces homicide rates.
> **Data**: Countries with stricter gun laws have lower homicide rates.
> **Warrant**: What happens in other countries is a reliable predictor of what would happen here.
> **Backing**: The compared countries share similar economic and cultural conditions.
> **Rebuttal**: Some countries with strict laws still have high homicide rates, but those cases involve confounding factors (e.g., drug cartel violence).
> **Qualifier**: In most cases, stricter gun control correlates with reduced homicides.

### 写作中的实际应用

不必在文章中标出这六个标签，但要在写作前确认每个要素都有对应内容，尤其是 **Warrant** 和 **Rebuttal**——这两项是区分优秀与平庸论证的关键。

- 没有 Warrant → 数据和主张之间是断桥，读者无法自动完成推理。
- 没有 Rebuttal → 文章显得一面之词，在 GRE 和学术写作中是硬伤。

## 三、两种模型的选用指南

| 场景 | 推荐模型 | 原因 |
|------|----------|------|
| 托福 Independent Writing | 五段式 | 时间紧，结构清晰优先 |
| 雅思 Task 2 | 五段式 + rebuttal 段落 | 评分标准要求 address both sides |
| GRE Issue | 五段式 + 强 rebuttal | 需要体现辩证思维 |
| GRE Argument | Toulmin | 本身就是找逻辑漏洞 |
| 本科 essay | 五段式 | 多数教授期望的标准格式 |
| 研究生论文 | Toulmin 或混合 | 需要更精细的论证结构 |
